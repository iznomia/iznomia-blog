## 前置知识

[行列式求值](https://www.luogu.com.cn/problem/P7112)，枚举主对角线上的位置，通过行消法变换 + 辗转相除将一个行首转化成 $0$，然后换法变换将这一列底下的东西转成 $0$。

```cpp
int sol(void) {
    int f = 1;
    for (int i = 1; i <= n; ++i)
        for (int j = i + 1; j <= n; ++j) {
            while (a[i][i]) {
                int d = a[j][i] / a[i][i];
                for (int k = i; k <= n; ++k)
                    a[j][k] = (a[j][k] - 1ll * d * a[i][k] % P + P) % P;
                swap(a[i], a[j]), f = -f;
            }
            swap(a[i], a[j]); f = -f;
        }
    int ans = f;
    for (int i = 1; i <= n; ++i) ans = 1ll * ans * a[i][i] % P;
    return (ans + P) % P;
}
```
